The difference

Tiles now sit on the path — they're waypoints along a single flowing line, just like the reference image. The path is one continuous bezier curve that bows and curves as it travels across the map.

Small anchor dots mark where each tile connects to the path. In the real app, Imagen 3 generates the background to complement whatever path layout the algorithm picks for that trip.
