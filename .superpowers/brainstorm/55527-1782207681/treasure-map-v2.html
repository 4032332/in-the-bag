What changed

  • Tiles are now light parchment with dark brown text — much easier to read against the map
  • Each tile has a slight random rotation to feel hand-placed
  • The dotted path uses smooth cubic bezier curves, winding left-right-left-right in a natural S-shape
  • Path doubles back on itself between days — organic, not rigid
  • Illustrations (Fuji, Torii, Pagoda, cherry blossom) are subtler and larger-scale
  • Compass rose, decorative corner borders added for authenticity

In the real app

  • Gemini Imagen 3 generates the full background illustration — this is a CSS approximation
  • The map style adapts per destination (pirate/tropical for Hawaii, samurai/feudal for Japan, etc.)
  • Tile colours and border style could also adapt to the destination theme
